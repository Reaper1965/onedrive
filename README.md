## 🎯 TL;DR

Showcase, share, preview, and download files inside *your* OneDrive with Chiheisen-Vercel-Index -

- 💸 Completely free to host
- ⚡ Super fast and responsive
- ⏱️ Takes less than 15 minutes to setup
- ⚒️ Highly customisable
- 🍌 More importantly, we are pretty (●'◡'●)

---

## 🚀 Getting Started!

1. Fork the repo [Chiheisen-Vercel-Index](https://github.com/Chiheisen-Union/Chiheisen-Vercel-Index)
2. Change `userPrincipleName` in `site.config.js` to your Microsoft account email address
3. Change `baseDirectory` in `site.config.js` to a folder under your OneDrive root directory
4. Optional - Check if you need to change `api.config.js`
5. Import to Vercel and add an Upstash integration, which imports REDIS_URL env var to Vercel
6. Trigger a redeploy, and profit! 🎉
7. The deployed Chiheisen-Vercel-Index will guide you through an authorisation process, and store the required tokens in the Redis database

> 📚 For more detailed information, follow our [Deployment Guide](https://github.com/Chiheisen-Union/Chiheisen-Vercel-Index/wiki/Deployment-Guide)

## ✨ Features

<div align="center">

| Feature | Description |
|---------|-------------|
| 👀 **File Preview** | PDF, EPUB, markdown, code, plain text |
| 💠 **List/Grid Layouts** | For previewing images and documents with thumbnails |
| 🎥 **Media Playback** | mp4, mp3, with subtitles support |
| 📄 **Office Preview** | docx, pptx, xlsx support |
| 📝 **README Preview** | Renders code blocks, images with relative links |
| 📑 **Pagination** | For folders with 200+ items |
| 🔒 **Protected Folders** | Password protected routes and files |
| ⏬ **Multi-file Download** | Compress and download multiple files/folders |
| 🔎 **Native Search** | Search through shared OneDrive files |

</div>

### Additional Features

- 🚀 Streamlined deployment, without manual token management
- 📤 Direct raw-file serving and hosting
- 🌙 Full dark mode support
- 🎨 Extensive style and website customizations

> **Note**: This project focuses on showcasing and providing a way for others to download files from your OneDrive. Emphasis on **free** and **serverless**. If you need WebDAV or use other cloud providers, check out [alist](https://github.com/alist-org/alist).
